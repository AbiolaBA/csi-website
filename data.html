cii — the CSI Python package

All pipeline modules + config.yaml + requirements.txt

Download
# 1. install
pip install -r requirements.txt

# 2. fetch the Atlas data vintage (recorded in a manifest)
python -m cii.download

# 3. build everything — panels, rankings, diagnostics
python -m cii.run

The heart of the index, in code

# capability.py — Step 1 (v0.3, complexity-aware)
df["density"] = (1.0 - df["distance"]).clip(0.0, 1.0)
pp["simplicity"] = 1.0 - pp.groupby("year")["pci"].rank(pct=True)
df["cap"] = df[["density", "simplicity"]].max(axis=1)

# index.py — Steps 4–5
df["contrib"] = df["w"] * df["cap"] * df["dep"]   # w · g
panel["CSI_raw"] = 1.0 - panel["CCG"]
panel["CSI"] = (panel["CSI_raw"] - grp.transform("mean")) / grp.transform("std")

Provenance

Data sources

The lab computes on published, public data only — and records the exact vintage of every input.

Atlas of Economic Complexity

Harvard Growth Lab, via Harvard Dataverse. Country-product-year trade with published distance, RCA and PCI. HS92 (1995–) and SITC Rev.2 (1962–). The Tier-1 backbone.

BACI

CEPII's harmonised bilateral trade at HS6, with quantities — the unit-value source behind CSI-Preference.

UNIDO INDSTAT

Industrial production by ISIC industry — the domestic-production input to Tier 2's apparent-consumption estimator.

World Development Indicators

Income, population, manufacturing, governance-adjacent controls; green-growth outcome variables.

UN BEC Rev.5 (logic)

The end-use classification framework guiding the in-project consumption-goods set — published in full for review.

Concordances

HS92↔ISIC Rev.3 and HS↔SITC concordances used to place trade and production on a common axis.
